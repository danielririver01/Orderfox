# Orderfox 🦊

Motor ligero de pedidos para negocios locales.
Optimizado para conexión móvil y flujo por WhatsApp.

## Stack
- Flask
- Jinja
- MySQL / MongoDB (pendiente decisión)

## Objetivo MVP
- Autenticación del negocio
- Gestión de categorías y productos
- Menú público
- Pedido vía WhatsApp

Si no está en esta lista, no existe.
