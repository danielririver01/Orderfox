# Orderfox

## Descripción

Orderfox es un motor de pedidos diseñado para SaaS de comida rápida.
Su objetivo principal es permitir la creación rápida y eficiente de pedidos,
optimizando la experiencia móvil, el manejo de categorías obligatorias
y la venta de extras como fuente clave de ingresos.

El sistema está pensado para operar con múltiples restaurantes,
respetando el aislamiento de datos por negocio y facilitando
la escalabilidad del producto.

## Capacidades principales

- Creación rápida de pedidos optimizada para dispositivos móviles
- Gestión de productos con categorías obligatorias
- Sistema de extras configurable por restaurante
- Aislamiento de datos por restaurante (multi-tenant)
- Base sólida para integración con WhatsApp y canales externos
- Arquitectura modular preparada para escalar como SaaS

## Caso de uso

Orderfox permite a restaurantes de comida rápida recibir y gestionar pedidos
de forma simple y veloz, reduciendo fricción en el proceso de compra
y aumentando el ticket promedio mediante extras bien definidos.
